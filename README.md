# Godot Roguelike Template

A Godot Engine template project designed to kickstart the development of traditional turn-based roguelike games. This template provides the foundational systems and mechanics needed for exploration, combat, level progression, inventory management, field of view (FOV), dungeon generation, and more. The goal is to enable developers to quickly prototype and build their own unique roguelike experiences.

## Features

- **Turn-based gameplay**: Classic roguelike turn system for player and enemies.
- **Procedural dungeon generation**: Easily customizable algorithms for creating unique levels.
- **Field of View (FOV)**: Visibility and exploration mechanics.
- **Basic combat system**: Melee and ranged combat foundations.
- **Inventory management**: Item pickup, usage, and equipment.
- **Level progression**: Experience, leveling, and stat growth.
- **Modular entity system**: Easily extendable for new monsters, items, and features.
- **Godot 4.x ready**: Built for the latest stable Godot version.

## Getting Started

1. **Clone this repository**
2. Open the project in Godot Engine (4.x recommended)
3. Run the project or start building your own roguelike!
